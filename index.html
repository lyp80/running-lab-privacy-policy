当前版本说明

当前测试版本无需注册即可浏览基础赛事信息，不接入广告、用户行为分析、设备健康平台或云端运动数据同步。赛事地图仅在您主动同意后加载高德地图 SDK；当前不会申请系统定位权限，也不会获取您的实时位置。

Running Lab 项目（以下简称“我们”）重视您的个人信息与隐私。本政策适用于 Running Lab 移动应用及我们明确标注适用本政策的相关服务。

在使用涉及地图、运动数据或其他个人信息的功能前，请阅读本政策。您可以拒绝非必要信息处理，并继续使用不依赖该信息的基础功能。

01适用范围

本政策说明 Running Lab 当前版本的个人信息处理活动，包括赛事地图、隐私偏好以及您主动使用的跑者分析功能。

佳明、华为、COROS、Strava、Apple Health、Health Connect、ITRA 数据连接、账号系统和云端同步仍属于规划能力，当前版本尚未接入。相关功能正式上线前，我们将更新本政策，并在需要时另行取得您的授权或单独同意。

02我们处理的信息

2.1 本地隐私偏好

当您选择同意、拒绝或撤回赛事地图授权时，我们会在您的设备本地保存该选择，用于决定是否加载高德地图。该记录目前不上传至 Running Lab 服务器；卸载应用、清除应用数据或在应用内重新选择后，记录会被删除或更新。

2.2 您主动提供的跑步信息

当相关功能开放后，您可以主动输入姓名或昵称、ITRA ID、全马或半马成绩、目标赛事、目标时间及主观能力描述，以生成跑力画像、完赛预测和分段策略。我们仅为提供您选择的分析功能处理这些信息，不会用于广告定向。

当前版本没有账号注册和云端同步。任何需要上传、长期保存或跨设备同步的功能上线前，我们会在功能页面明确说明处理目的、信息种类和保存期限。

2.3 公开赛事资料

Running Lab 会整理赛事名称、时间、公开赛道 GPX、检查点、距离、爬升、下降、公开完赛成绩和天气等赛事资料。这类资料主要用于赛道分析与策略模型；如其中包含可识别跑者的信息，我们将按适用法律和数据来源规则处理，并提供异议与删除渠道。

03敏感个人信息

运动轨迹可能构成行踪轨迹信息；心率、训练负荷、健康平台数据等可能构成医疗健康相关信息。此类信息一旦泄露或被不当使用，可能对您的人身或财产安全产生影响。

当前版本不会读取您的设备运动轨迹、实时位置或健康平台数据。未来启用相应功能时，我们将遵循最小必要原则，在具体场景中说明必要性和可能影响，并依法取得单独同意。个人上传轨迹将默认设为私密，公开前会提供起终点隐私区域等保护选项。

05第三方 SDK

为展示正式赛事的位置和赛道轨迹，Running Lab 集成以下第三方 SDK。我们仅在您阅读相关政策并主动同意、且进入地图功能场景后初始化该 SDK。

SDK 名称
高德开放平台地图 SDK（合包）
提供方
北京高德图强科技有限公司
使用目的
展示赛事地图、赛事点位和 GPX 赛道轨迹
处理方式
SDK 在本机采集并根据服务需要进行网络传输

高德 SDK 可能处理的信息

经纬度；IP 地址、GNSS 信息、网络类型、Wi-Fi 状态及参数、Wi-Fi 列表、SSID、BSSID、基站信息、信号强度和传感器信息；IDFA、OAID；应用名称与版本；设备品牌、型号、操作系统、运营商和屏幕分辨率等设备及系统信息。实际处理范围会因设备、系统和我们配置的权限而不同。

Running Lab 当前关闭地图的用户定位能力，不申请系统位置、后台位置或 Wi-Fi 状态权限。赛事 GPX 坐标属于赛事资料，不是从您的设备定位获得的个人轨迹。

06共享、转让与公开

我们不会出售您的个人信息，也不会为了广告定向向第三方共享您的跑步、健康或轨迹数据。

  • 仅在您同意并使用赛事地图时，按本政策所述由高德地图 SDK 处理相关信息。
  • 如未来需要委托云服务商保存账号或运动数据，我们会披露受托方类别、处理目的及保护措施，并监督其处理活动。
  • 发生合并、分立、解散或业务转让时，我们会依法告知接收方信息，并要求接收方继续遵守本政策；处理目的或方式发生变化时，将重新取得您的同意。
  • 除法律法规要求或为保护人身、财产安全等法定情形外，我们不会公开披露您的个人信息。

07保存与安全

我们仅在实现相应目的所需的最短期限内保存个人信息。当前地图授权偏好保存在您的设备本地，直至您撤回、清除应用数据或卸载应用。高德 SDK 处理信息的保存规则以其隐私政策为准。

我们将通过访问控制、最小权限、传输保护、版本记录和安全审查等措施保护个人信息。互联网服务无法保证绝对安全；如发生或可能发生个人信息泄露、篡改或丢失，我们将依法采取补救措施并履行通知义务。

未来启用云端存储、跨境传输或境外服务前，我们会另行完成合规评估并更新本政策。

08您的权利

在适用法律规定的范围内，您有权知情、决定、限制或拒绝个人信息处理，并可请求查阅、复制、更正、补充或删除您的个人信息。

  • 撤回地图同意：进入 Running Lab 的“我的 → 隐私与权限”，选择重新设置或撤回；撤回后不再加载高德地图。
  • 删除本地记录：可通过应用内设置、系统清除应用数据或卸载应用删除当前本地偏好。
  • 赛事成绩异议：如公开赛事资料中的姓名、成绩或身份匹配有误，可通过本政策末尾的渠道联系我们核验、修正或删除。
  • 账号注销与数据导出：账号系统上线时，我们会提供可操作的注销和导出入口，并说明依法不能立即删除的信息及其期限。

为保障安全，我们可能需要核验请求人与相关信息主体的身份。我们不会要求您在公开渠道提交完整轨迹、证件或健康资料。

09未成年人保护

Running Lab 当前面向具备相应民事行为能力的跑步爱好者，不专门面向不满 14 周岁的儿童。若我们发现未经监护人同意处理了儿童个人信息，将在核实后尽快删除或匿名化处理。

未来如提供儿童可使用的服务，我们将制定专门的儿童个人信息处理规则，并取得监护人同意。

10政策更新

随着账号、设备连接、轨迹上传、训练分析和云端同步等能力上线，我们可能更新本政策。更新后的版本会在本页面标明版本号与生效日期。

对处理目的、方式、信息种类或敏感个人信息处理产生重要影响的变更，我们会通过应用内弹窗或其他显著方式告知，并在法律要求时重新取得您的同意。未经您同意，我们不会削减您在本政策下享有的权利。
